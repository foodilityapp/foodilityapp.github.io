  • 2020/2/9

    • Hi Foodility users!

    We have been working on a new version that would now be shipped with an option for a Premium subscripion. It is not live on the app store yet, but it will be in a few days. We are currently thinking of a monthly price of $2.99 USD, and a yearly price of $29.99 USD.

    Some of the features that have already been implemented in our code for Premium are the following.
    - PDF Export with range, and combined export, with photos
    - Customizable Presets
    - Customizable Tag for General Note
    - Customizable Dot Color for Calendar

    Additional feature that will be free for everyone
    - Optional calorie

    We are also fixing a bug where posting a Weight post couldn't upload photos.

    Thank you very much for your support, and if you have any feedback on any of the upcoming changes, please feel free to email us! We would like to listen to you before making this update, and if you have any other features that you think would like to be included, please let us know!
